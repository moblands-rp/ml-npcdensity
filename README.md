# ml-npcdensity

Fully configurable NPC density reducer for QBCore FiveM servers.

## Features
- Reduce ambient NPCs (peds) instantly
- In-game menu with presets + custom value
- Configurable command name, default density, admin-only mode
- Works with qb-menu & qb-input

## Installation
1. Put the folder as `ml-npcdensity`
2. Add `ensure ml-npcdensity` to server.cfg
3. Restart server

## Configuration
Edit `config.lua` – everything is explained there.

## Commands
`/npcdensity` (or whatever you set in config)
